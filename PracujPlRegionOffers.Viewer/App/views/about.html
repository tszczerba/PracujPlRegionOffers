Zadanie

Treść otrzymanego zadania:

Proszę napisać program zliczający i zapisujący do archiwum co 5 minut ze strony pracuj.pl liczbę ofert pracy w podziale na województwa. Dozwolone jest użycie dowolnych narzędzi.

Użyte narzędzia i technologie

Obie aplikacje napisałem przy uzyciu Microsoft visual Studio 2013 Ultimate (90 dniowy trial)

Użyte technologie i frameworki

  • .Net Framework 4.5
  • Entity Framework 6.1 z SQL Compact 4.0
  • ASP.NET MVC z Web Api
  • AngularJS
  • Html i css, części Flat Ui, Responsive Grid System

Architektura

Rozwiązanie składa sie z czterech projektów:

  • Konsolowa aplikacja, która pobiera dane ze strony pracuj.pl w regularnych odstępach czasu i zapisuje je do bazy danych
  • Aplikacja web, która odczytuje dane z bazy w regularnych odstępach czasu i prezentuje na stronie
  • Projekt Commons - wspólne dla obu aplikacji klasy
  • Projekt Tests

Pliki do pobrania:

  • GitHub: tomaszszczerba/PracujPlRegionOffers
  • Visual Studio Solution: PracujPlRegionOffers.zip
  • Log aplikacji web: log.svclog
  • Log aplikacji konsolowej: log.svclog
  • Baza danych (można przeglądać compactview): RegionOffersDb.sdf

Uwagi:

  • Logi obu aplikacji znajdują się w plikach log.svclog w folderach uruchomieniowych
  • Ustawienia obu aplikacji znajdują sie w plikach App.config oraz Web.config.
  • Strona internetowa jest zoptymalizowana pod przeglądarkę Chrome
  • Przed uruchomieniem należy ustawić odpowiednio connection stringi tak aby w obu aplikacjach wskazywały na jedną bazę danych.
  • Jezeli aplikacja uruchamiana jest na nowej bazie danych należy uruchomić procedurę Add-Migration (nadając nazwę migracji np. Init ) a następnie Update-Database w Package Manager Console w celu zainicjalizowania i spopulowania jej odpowiednimi wpisami dla wymiaru Region.
  • W razie problemów z inicjalizacją bazy danych w katalogu \PracujPlRegionOffers\PracujPl.Commons\Database znajduje się zainicjalizowana baza danych.
  • Connection Stringi są skonfigurowane do działania na IIS Express w kaloguC:\PracujPlRegionOffers
